## Run Locally

Clone the project

```bash
  git clone https://github.com/nikolai-dimitrov/Figma-Designs-Implementation.git
```

Go to the project directory

```bash
  cd Figma-Designs-Implementation
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
